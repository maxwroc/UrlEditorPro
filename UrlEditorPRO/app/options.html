Redirections

You can create redirection rules which can be applied automatically (whenever browser tries to navigate to specific url).
Use cases examples:
  • Your test framework generates URL which contains a lot of redundant params and you want to get rid of them
  • You want to redirect all localhost requests to some other host
  • You want to navigate to some other website passing parameters from he current url
